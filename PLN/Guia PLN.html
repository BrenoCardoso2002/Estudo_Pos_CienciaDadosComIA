🚀 Guia Completo de PLN

O que é PLN?

Área da IA que permite computadores entenderem, interpretarem e gerarem linguagem humana.

Pré-processamento (Regex & Unidecode)

O que é: A limpeza inicial e padronização dos dados brutos.
Para que serve: Remove ruídos (links, emojis, acentos) que podem confundir o modelo, garantindo que a entrada seja limpa. 
import re
from unidecode import unidecode

def limpar_texto(texto):
    texto = unidecode(texto.lower())
    texto = re.sub(r'http\S+|www\S+|https\S+', '', texto)
    return re.sub(r'[^a-z\s]', '', texto).strip()

print(limpar_texto("Café! no site https://google.com ☕"))

Tokenização

O que é: O processo de quebrar uma frase em pedaços menores chamados tokens.
Para que serve: Permite que o computador analise o texto palavra por palavra (ou símbolo por símbolo) de forma estruturada. 
import nltk
from nltk.tokenize import word_tokenize

# Você só precisa baixar isso uma vez
nltk.download('punkt')

texto = "Bom dia, como vai?"
tokens = word_tokenize(texto)

print(tokens)

Resultado esperado:

['Bom', 'dia', ',', 'como', 'vai', '?']

Stop Words

O que é: Palavras muito frequentes que não carregam o sentido principal (ex: "e", "o", "de").
Para que serve: Filtrar essas palavras ajuda o modelo a focar apenas nos termos que realmente definem o contexto do texto. 
import nltk
from nltk.corpus import stopwords

# Baixa a lista de palavras irrelevantes
nltk.download('stopwords')

stop_words = set(stopwords.words("portuguese"))
tokens = ['eu','quero','pagar','a','conta','de','luz']

stop_words.add("pagar") # Se "pagar" não for importante para você

filtrados = [w for w in tokens if w not in stop_words]
print(filtrados)

Resultado esperado:

['quero', 'conta', 'luz']

Bag of Words

O que é: Uma forma de transformar texto em vetores baseados na contagem de ocorrência de cada palavra.
Para que serve: Cria uma representação numérica simples que algoritmos de machine learning conseguem processar matematicamente. 
from sklearn.feature_extraction.text import CountVectorizer

# Seus textos (o "corpus")
documentos = [
    "eu quero pagar a conta",
    "a conta de luz",
    "quero luz na conta"
]

# Criamos o "vetorizador"
vectorizer = CountVectorizer()

# O comando fit_transform aprende as palavras e já conta as frequências
X = vectorizer.fit_transform(documentos)

# Mostra as palavras que ele identificou como colunas
print(vectorizer.get_feature_names_out())

# Mostra a contagem em formato de matriz
print(X.toarray())

TF-IDF

O que é: Uma métrica que pondera a frequência de uma palavra em um texto contra sua raridade no conjunto total.
Para que serve: Identifica palavras que são realmente importantes e específicas de um documento, ignorando termos genéricos. 
TF-IDF = TF × IDF

TF = frequência da palavra no documento
IDF = log(N / df)

N = total de documentos
df = documentos que contêm a palavra

Word Embeddings (Significado Vetorial)

O que é: Uma técnica onde palavras são representadas por vetores densos em um espaço multidimensional.
Para que serve: Permite que o computador entenda a semântica (o sentido). Palavras parecidas ficam com vetores matematicamente próximos. 
from sentence_transformers import SentenceTransformer
# Modelo multilingue (PT-BR)
model = SentenceTransformer('paraphrase-multilingual-MiniLM-L12-v2')

frases = ["O dia está lindo", "O tempo está belo"]
embeddings = model.encode(frases)
# frases com significados similares terão vetores próximos

Lemmatização vs Stemming

O que são: Métodos para reduzir palavras às suas formas raízes.
Para que servem: Normalizam o vocabulário (ex: "correndo" e "correu" viram "correr"), facilitando a contagem de ideias e não apenas de flexões gramaticais.

Lemmatização

import spacy

nlp = spacy.load("pt_core_news_sm")

doc = nlp("correndo correu correram")

lemmas = [token.lemma_ for token in doc]

print(lemmas)

Stemming

import nltk   
from nltk.stem import RSLPStemmer

stemmer = RSLPStemmer()

palavras = ["correndo", "correu", "correram"]

stems = [stemmer.stem(p) for p in palavras]

print(stems)

Whisper (Áudio → Texto)

O que é: Um sistema automático de reconhecimento de fala treinado em dados massivos.
Para que serve: Transcrever áudios para texto com precisão profissional, suportando tradução e vários idiomas nativamente.

Escolha do modelo:

  • tiny/base → rápido, menos preciso
  • medium → melhor custo-benefício
  • large → máxima precisão

Reescrita de Texto com IA

O que é: Utilização de modelos de linguagem (LLMs) via API otimizada para velocidade extrema.
Para que serve: Automatizar tarefas complexas de escrita, como mudar o tom de um e-mail ou resumir parágrafos rapidamente. 
from groq import Groq

client = Groq(api_key="SUA_CHAVE")

def reescrever_texto(texto, modo):
    prompt = f'''
Reescreva o texto abaixo no estilo {modo}:

- formal
- informal
- neutro

Texto: {texto}
'''

    resposta = client.chat.completions.create(
        model="llama3-8b-8192",
        messages=[{"role": "user", "content": prompt}]
    )

    return resposta.choices[0].message.content

print(reescrever_texto("mano preciso de ajuda", "formal"))
import google.generativeai as genai

genai.configure(api_key="SUA_CHAVE_GEMINI")
model = genai.GenerativeModel('gemini-1.5-flash')

def reescrever_texto(texto, modo):
    prompt = f"Reescreva o texto abaixo no estilo {modo}: {texto}"
    
    resposta = model.generate_content(prompt)
    return resposta.text

print(reescrever_texto("mano preciso de ajuda", "formal"))

Pipeline de CSV com IA (Produção)

O que é: Um fluxo de trabalho automatizado para processar bases de dados em lote.
Para que serve: Processar milhares de linhas de um arquivo CSV de uma vez, utilizando a IA para preencher lacunas, categorizar ou resumir informações. 
import pandas as pd
from groq import Groq
import time
import os
from tqdm import tqdm # Para barra de progresso real

# 1. Configuração do Cliente
client = Groq(api_key=os.environ.get("GROQ_API_KEY"))

# 2. Carregamento dos dados
try:
    base_completa = pd.read_csv("base_completa.csv")
    base_faltante = pd.read_csv("base_faltante.csv")
except FileNotFoundError:
    print("Erro nos arquivos CSV.")
    exit()

# 3. Construção dos exemplos (Few-Shot)
exemplos = ""
for _, row in base_completa.head(5).iterrows():
    exemplos += f"\nTexto: {row['A']}\nCategoria: {row['B']}\nResumo: {row['C']}\n---"

# 4. Função para consultar a IA
def completar_texto(texto):
    prompt = f'Use estes exemplos: {exemplos}\nTexto: {texto}\nGere: Categoria e Resumo.'
    try:
        resposta = client.chat.completions.create(
            model="llama-3.1-8b-instant",
            messages=[{"role": "user", "content": prompt}],
            temperature=0.1
        )
        return resposta.choices[0].message.content
    except Exception as e:
        return f"Erro: {e}"

# 5. Loop de processamento com tqdm
print(f"Processando {len(base_faltante)} linhas...")
tqdm.pandas() 

base_faltante['Resposta_IA'] = base_faltante['A'].progress_apply(completar_texto)

# 6. Salvar o resultado final em CSV
base_faltante.to_csv("resultado_final.csv", index=False, encoding='utf-8-sig')

import pandas as pd
import google.generativeai as genai
import time
import os
from tqdm import tqdm

genai.configure(api_key=os.environ.get("GOOGLE_API_KEY"))

model = genai.GenerativeModel('gemini-1.5-flash')

# 2. Carregamento dos dados
try:
    base_completa = pd.read_csv("base_completa.csv")
    base_faltante = pd.read_csv("base_faltante.csv")
except FileNotFoundError:
    print("Erro: Arquivos CSV não encontrados.")
    exit()

# 3. Construção dos exemplos (Few-Shot)
exemplos = ""
for _, row in base_completa.head(5).iterrows():
    exemplos += f"\nTexto: {row['A']}\nCategoria: {row['B']}\nResumo: {row['C']}\n---"

# 4. Função para consultar a IA
def completar_texto(texto):
    prompt = f'Use estes exemplos como guia:\n{exemplos}\nTexto: {texto}\nGere apenas a Categoria e o Resumo no mesmo formato dos exemplos.'
    
    try:
        # No Gemini, usamos generate_content
        response = model.generate_content(
            prompt,
            generation_config=genai.types.GenerationConfig(
                temperature=0.1,
            )
        )
        return response.text.strip()
    except Exception as e:
        # Em caso de erro de cota (Rate Limit), você pode adicionar um time.sleep aqui
        return f"Erro: {e}"

tqdm.pandas()
